# jokenpo em python
Projeto simples de Jokenpo em Python para praticar lógica de programação.

## Funcionalidades

- Escolha entre pedra, papel e tesoura
- Jogada aleatória do computador
- Verificação automática do vencedor
- Placar com vitórias, derrotas e empates
- Opção de encerrar o jogo digitando `sair`

## Tecnologias utilizadas

- Python 3

## Como executar

```bash
py jokenpo.py
```
