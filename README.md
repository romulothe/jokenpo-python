# Jokenpô em Python

Projeto simples desenvolvido em Python para praticar lógica de programação por meio de um jogo de Jokenpô executado no terminal.

O objetivo do projeto é aplicar conceitos básicos da linguagem, como entrada de dados, estruturas condicionais, repetição, funções e geração de jogadas aleatórias.

## Funcionalidades

- Escolha entre pedra, papel e tesoura.
- Jogada aleatória do computador.
- Verificação automática do vencedor.
- Exibição de vitória, derrota ou empate.
- Opção para encerrar o jogo digitando `sair`.

## Tecnologias utilizadas

- Python 3

## Como executar

Clone o repositório:

```bash
git clone https://github.com/romulothe/jokenpo-python.git
```

Acesse a pasta do projeto:

```bash
cd jokenpo-python
```

Execute o arquivo principal:

```bash
py jokenpo.py
```

Em alguns ambientes, também é possível executar com:

```bash
python jokenpo.py
```
